#### Mengecilkan Ukuran File PDF <a name="Resize-PDF"></a>

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Jika sudah selanjutnya klik `Select File` untuk mengambil file pdf yang akan kita kurangi ukurannya atau size file PDF
